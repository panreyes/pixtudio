Definition
----------

**INT** rm ( &lt;**STRING** filename&gt; )

Removes (deletes) the file specified with *filename*.

Parameters
----------

  --------------------- -------------------------------------------------
  **STRING** filename   - The name of the file to be removed (deleted).
  --------------------- -------------------------------------------------

Returns
-------

**INT** : Success

  ------------------------------- --------------------------------------------------------
  0 ([false](false "wikilink"))   - Removing the file with the specified name failed.
  !0 ([true](true "wikilink"))    - Removing the file with the specified name succeeded.
  ------------------------------- --------------------------------------------------------
