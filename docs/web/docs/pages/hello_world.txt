The Bennu Hello World! Example.

    import "mod_say" // import the module to output text to console, using say()

    Process Main() // start the definition of the main process
    Begin // start the code
        say("Hello World!");
    End // end the definition of the main process

Used in example: [import](import "wikilink"),
[Process](Process "wikilink"), [Begin](Begin "wikilink"),
[End](End "wikilink"), [say](say "wikilink")()

This will import the module "*mod\_say*" which contains the definition
of the "*say()*" function. The function will echo the argument "Hello
World" to the screen.
