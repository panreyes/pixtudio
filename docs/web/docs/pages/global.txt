Syntax
------

**Global**

:   \[ <global variables> \]

**End**

Description
-----------

Global is a reserved word used to initiate the declaration of [global
variables](:Category:global_variables "wikilink"). Terminating the
declaration block with an [End](End "wikilink") is needed when the
Global is not used in conjunction with the main code of the
[Program](Program "wikilink").

For a list of predefined global variables, see [this
page](:Category:global_variables "wikilink").

Example
-------

    Global // Declare global variables here
    End

    Process Main()
    Begin
    End

    Global // Declare global variables here
    End
