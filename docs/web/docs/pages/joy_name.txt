Syntax
------

**STRING** joy\_name ( &lt;**INT** JoyID&gt; )

Description
-----------

Returns the name of the specified [joystick](joystick "wikilink"). This
is a string describing the specified joystick, mostly used for the brand
and model of the joystick.

Parameters
----------

  --------------- -------------------------------------------------------------------------
  **INT** JoyID   - The [JoyID](JoyID "wikilink") of the [joystick](joystick "wikilink").
  --------------- -------------------------------------------------------------------------

Returns
-------

**STRING** : The name of the [joystick](joystick "wikilink").
