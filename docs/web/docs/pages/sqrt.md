Definition
----------

**FLOAT** sqrt ( &lt;**FLOAT** value&gt; )

Returns the square root of a certain value.

Parameters
----------

  ----------------- --------------------------------------------------------
  **FLOAT** value   - The value of which the square root will be returned.
  ----------------- --------------------------------------------------------

Returns
-------

**FLOAT** : The square root of *value*.
