##Blit flags
###Description
Blit flags are bit flags which can be passed to some map functions, to specify a certain effect when blitting a graphic. These functions are:
- xput()
- map_xput()
- map_xputnp()

They can also be used to specify a certain effect for the blitting of the graphic of a process, by assigning blit flags to its local variable flags.

###List
<table class="table table-bordered table-hover table-condensed table-striped">
	<tr><th>Constant<th>Value<th>Description
	<tr><td>B_HMIRROR<td>1<td>Blit the graph horizontally mirrored.
	<tr><td>B_VMIRROR<td>2<td>Blit the graph vertically mirrored.
	<tr><td>B_TRANSLUCENT<td>4<td>Blit the graph with half transparency.
	<tr><td>B_ALPHA<td>8<td>Blit the graph in some way. (What does this do exactly?)
	<tr><td>B_ABLEND<td>16<td>Blit the graph using additive blending (nice effect for fire).
	<tr><td>B_SBLEND<td>32<td>Blit the graph using subtractive blending (nice effect for ghosting).
	<tr><td>B_NOCOLORKEY<td>128<td>Blit the transparent parts of the graph as black (color 0).
</table>
