Description
-----------

A parameter is the [variable](variable "wikilink") and its
[value](value "wikilink") inside the definition of a
[function](function "wikilink") or [process](process "wikilink") that is
received from the [calling](function_call "wikilink") environment. The
value passed on when calling the function or process is called an
[argument](argument "wikilink").

If you use the same name as a [local
variable](local_variable "wikilink") for a parameter, it will not make a
new variable, but use the local variable instead. This means that passed
arguments will modify the local variable. This is commonly used for
[x](x "wikilink"), [y](y "wikilink") and [graph](graph "wikilink").

Notes
-----

There currently is a limit of 15 parameters that can be used per
function or process.

Example
-------

    Process Main()
    Private
        int argument = 3;
    Begin
        my_proc( argument );
    End

    Process my_proc( int parameter )
    Begin
        //statements
    End
