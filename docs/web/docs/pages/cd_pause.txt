Definition
----------

**INT** cd\_pause ( &lt;**INT** cdID&gt; )

Pauses playback of the specified CD/DVD drive.

Parameters
----------

  -------------- ------------------------------------------------------
  **INT** cdID   - The cdID of the CD/DVD drive to pause playback on.
  -------------- ------------------------------------------------------

Returns
-------

**INT**

  --- -------------------------------------------
  0   - Invalid CD/DVD drive.
  1   - Playback of the CD/DVD drive is paused.
  --- -------------------------------------------
