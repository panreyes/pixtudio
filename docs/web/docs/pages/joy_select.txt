Syntax
------

**INT** joy\_select ( &lt;**INT** JoyID&gt; )

Description
-----------

Select the joystick with id equals to JoyID. The old name
[select\_joy](select_joy "wikilink")() is deprecated.

Parameters
----------

  --------------- -------------------------------------------------------------------------
  **INT** JoyID   - The [JoyID](JoyID "wikilink") of the [joystick](joystick "wikilink").
  --------------- -------------------------------------------------------------------------

Returns
-------

**INT** : The ID of the selected joystick number.
