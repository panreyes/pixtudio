Definition
----------

**FLOAT** atof ( &lt;**STRING** str&gt; )

Returns the [float](float "wikilink") value of the number specified in a
certain [string](string "wikilink").

Parameters
----------

  ---------------- -----------------------------------------------------------------------------
  **STRING** str   - The string specifying a number of which the float value will be returned.
  ---------------- -----------------------------------------------------------------------------

Returns
-------

**FLOAT** : The float value of the number specified by the specified
string.
