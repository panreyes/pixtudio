Definition
----------

**STRING** ftoa ( &lt;**FLOAT** value&gt; )

Returns a [string](string "wikilink") containing a certain
[float](float "wikilink") value.

Parameters
----------

  -------------------- -----------------------------------------------
  '''FLOAT ''' value   - The value the returned string will contain.
  -------------------- -----------------------------------------------

Returns
-------

**STRING** : The string containing the specified value, including sign
and decimal point.
