Definition
----------

**INT** cd\_stop ( &lt;**INT** cdID&gt; )

Stops playback of the specified CD/DVD drive.

Parameters
----------

  -------------- -----------------------------------------------------
  **INT** cdID   - The cdID of the CD/DVD drive to stop playback on.
  -------------- -----------------------------------------------------

Returns
-------

**INT**

  --- --------------------------------------------
  0   - Invalid CD/DVD drive.
  1   - Playback of the CD/DVD drive is stopped.
  --- --------------------------------------------
