Definition
----------

**WaveID**

A WaveID is an identifier associated with a certain wave-sound loaded by
[load\_wav](load_wav "wikilink")(). The identifier can be used in
various [other functions](Functioncategory:Sound "wikilink"), like
[unload\_wav](unload_wav "wikilink")() to unload the wave-sound.
