Definition
----------

**INT** dump\_type = PARTIAL\_DUMP

Dump\_type is a [global variable](global_variable "wikilink"), holding
the current [dump\_mode](dump_modes "wikilink"). The mode can be changed
by assigning a different mode to the variable. Default is
[PARTIAL\_DUMP](PARTIAL_DUMP "wikilink").

See also [restore\_type](restore_type "wikilink").
