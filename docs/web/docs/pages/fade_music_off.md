Definition
----------

**INT** fade\_music\_off ( &lt;**INT** ms&gt; )

Fades off the song, played by [Play\_song](Play_song "wikilink")().

Parameters
----------

  ------------ ------------------------------------------
  **INT** ms   - Microseconds of fading (the duration).
  ------------ ------------------------------------------

Returns
-------

**INT** : Error.

  ---- --------------------------
  -1   - Error: sound inactive.
  ---- --------------------------

--------
