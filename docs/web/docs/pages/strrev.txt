Definition
----------

**STRING** strrev ( &lt;**STRING** str&gt; )

Returns a reversed version of a certain [string](string "wikilink"),
meaning the characters are in reversed order.

Parameters
----------

  ---------------- ----------------------------
  **STRING** str   - The non reversed string.
  ---------------- ----------------------------

Returns
-------

**STRING** : The reversed string.
