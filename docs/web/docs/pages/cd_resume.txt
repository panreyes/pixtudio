Definition
----------

**INT** cd\_resume ( &lt;**INT** cdID&gt; )

Resumes playback of the specified CD/DVD drive.

Parameters
----------

  -------------- -------------------------------------------------------
  **INT** cdID   - The cdID of the CD/DVD drive to resume playback on.
  -------------- -------------------------------------------------------

Returns
-------

**INT**

  --- --------------------------------------------
  0   - Invalid CD/DVD drive.
  1   - Playback of the CD/DVD drive is resumed.
  --- --------------------------------------------
