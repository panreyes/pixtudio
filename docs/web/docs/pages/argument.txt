Definition
----------

An argument is the [value](value "wikilink") passed on when calling a
[function](function "wikilink") or [process](process "wikilink"). The
[variable](variable "wikilink") and its value inside the definition of a
[function](function "wikilink") or [process](process "wikilink") is
called a [parameter](parameter "wikilink").

Example
-------

    Function int my_proc( int parameter )
    Begin
        //statements
        return 0;
    End

    Process Main()
    Private
        int argument = 3;
    Begin
        my_proc( argument );
        my_proc( 4);
    End

Used in example: [Function](Function "wikilink"),
[Process](Process "wikilink")
