Syntax
------

**FLOAT** asin ( &lt;**FLOAT** value&gt; )

Description
-----------

Returns the arcsine of a certain *value*.

This [function](function "wikilink") performs an arcsine calculation on
a certain value and returns an [angle](angle "wikilink") between and
including -90000 and 90000 (-90-90ยบ).

Parameters
----------

  ----------------- --------------------------------------------------------
  **FLOAT** value   - The value to be performed an arcsine calculation on.
  ----------------- --------------------------------------------------------

Returns
-------

**FLOAT** : The arcsine result of the specified value, an angle between
and including -90000 and 90000 (-90-90ยบ).

Notes
-----

The [angle](angle "wikilink") value returned by this function is in
thousandths of degrees, as most angles within [Bennu](Bennu "wikilink")
are.

To read about all aspects of trigonometry, you can visit Wikipedia's
[Trigonometric
function](http://en.wikipedia.org/wiki/Trigonometric_function) page.
