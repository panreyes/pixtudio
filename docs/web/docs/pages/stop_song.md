Definition
----------

**INT** stop\_song ( )

Stops a song loaded with [load\_song](load_song "wikilink")() and played
with [play\_song](play_song "wikilink")().

Parameters
----------

This function has no parameters.

Returns
-------

**INT** : Status

  --- ------------------------
  0   - The song is stopped.
  --- ------------------------
